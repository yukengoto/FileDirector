## 主な機能

- **動画・画像の一元管理** — フォルダに散らばったファイルをDBに登録、サムネイル一覧表示
- **タグで整理・絞り込み** — お気に入り、ジャンル、用途など自由にタグ付け
- **YouTube・SNSからダウンロード** — yt-dlp / gallery-dl 連携、完了後に自動登録
- **拡張子関連付け** — エクスプローラーからダブルクリックで直接再生
- **AI類似検索・自動分類** — フレーム画像またはキーワードで類似ファイルを検索、K-Meansで自動グループ分け
- **動画変換・音声抽出** — FFmpeg連携でカット・変換・抽出
- **Web UI** — スマホ・タブレットのブラウザからもアクセス可能

---

## 動作環境

- **Windows 10 / 11** (64-bit)
- メモリ 4GB 以上推奨

**外部ツール（オプション）**

| ツール | 用途 |
|--------|------|
| FFmpeg | 動画変換・サムネイル生成 |
| yt-dlp | YouTube等の動画ダウンロード |
| gallery-dl | Twitter・Pixiv等の画像ダウンロード |
| Python 3 | AI類似検索・自動分類（自動セットアップ） |
